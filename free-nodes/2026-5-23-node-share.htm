科学上网是指通过技术手段（如 VPN、代理服务器等）绕过网络限制，访问被屏蔽的网站和服务。其核心目的是突破地域限制，实现自由访问互联网。科学上网的主要应用场景包括：

  1. 访问被屏蔽的网站：如 Google、YouTube、Facebook 等。

  2. 保护隐私：通过加密技术，防止个人信息被窃取或监控。

  3. 使用外部服务：某些在线服务（如云存储、学术资源）可能因地域限制无法访问。

FAQ

Nekoray 支持 Trojan 吗？
支持。Nekoray 基于 sing-box 核心，能兼容 Trojan、Vmess、Vless、Hysteria2 等多种协议。用户可在同一界面中管理和切换不同类型节点，非常方便。
Netch 如何为游戏流量提供加速？
Netch 支持 UDP 转发，可为游戏流量建立高速通道。结合策略组和节点延迟检测，可减少丢包和延迟，保证游戏访问稳定流畅，提高网络体验。
V2Ray 的 ws+tls 传输模式有什么优点？
ws+tls 模式利用 WebSocket 伪装成普通 HTTPS 流量，使得防火墙难以区分代理与正常访问。该模式兼容性强，适合部署在支持 Nginx 或 Caddy 的 Web 服务器上。
V2Ray 如何通过策略组优化访问速度？
策略组允许用户将节点按延迟、地域或用途分组。结合路由规则，V2Ray 会自动选择最优节点处理流量，提升访问速度并降低延迟，尤其在访问海外网站时效果明显。
Clash 的自动测速功能如何设置？
Clash 支持定期测速节点延迟，用户可以设置测速频率和超时时间。测速结果会影响策略组的节点选择，从而自动切换至最优节点，提高访问速度和连接稳定性。
Netch 全局代理与规则代理如何选择？
全局代理会将所有流量通过节点转发，适合游戏或全局科学上网；规则代理只针对指定应用、IP 或端口走节点，可优化带宽使用和访问效率，根据需求灵活选择模式。
Mihomo 启动后 CPU 占用高，有哪些优化建议？
检查是否启用了大量日志或复杂脚本，关闭不必要的规则提供者与日志调试；在多核服务器上启用并发或调整线程数；更新到最新版本以获得性能优化，并监控是否有异常连接导致处理压力。
V2Ray QUIC 传输适合什么网络环境？
QUIC 基于 UDP 支持多路复用和零握手，降低延迟，提高吞吐量。适合高丢包、不稳定网络环境，可保证视频、游戏和即时通讯流量的稳定性和流畅体验。
Matsuri 日志太多如何调整日志级别？
在 Matsuri 或其内核配置中修改日志级别参数（如从 debug 降为 info 或 warning），同时限制日志轮转与最大文件大小。生产环境建议关闭 debug 日志以减少磁盘 I/O 与性能影响。
PassWall2 如何根据延迟自动切换节点？
PassWall2 会定期测试节点延迟，并结合策略组自动切换到延迟最低的节点。无需用户手动操作，即可保证网页、视频和游戏流量的高速稳定访问，提高整体网络体验。
