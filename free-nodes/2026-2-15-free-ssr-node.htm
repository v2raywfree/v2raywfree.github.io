在信息爆炸的互联网时代，科学上网已不再是极客的专属，而逐渐成为了越来越多普通用户的刚需。尤其是当我们试图访问被限制的网站、获取全球化信息、观看海外流媒体服务时，传统的手机或电脑端代理工具已无法满足全家设备共同上网的需求。

一、什么是梅林小宝？它为何如此受欢迎？

“梅林小宝”并非一个独立的软件，而是一款基于**华硕梅林固件（Asuswrt-Merlin）**开发的插件级科学上网管理工具，通常以“脚本+Web前端”的形式存在。它的核心价值，在于将科学上网功能集成到路由器层面，从根本上解决多设备、多终端接入的问题。

为什么选择梅林小宝？

  • 全局代理能力强：部署在路由器层，可为全家所有设备（手机、电视、平板、游戏机）提供稳定代理。

  • 界面友好、操作简单：相比手动配置代理或iptables，梅林小宝提供了简洁直观的可视化操作界面。

  • 插件管理方便：支持在线升级、配置导入导出、定时更新规则等自动化功能。

  • 规则分流灵活：支持GFW列表、国内外域名自动识别分流，有效提高访问效率。

  • 免费开源、社区活跃：有强大的开发者社区支持，教程丰富，问题解决迅速。

二、使用梅林小宝前的准备工作

你需要具备：

  1. 一台支持梅林固件的华硕或刷了梅林固件的路由器
    常见支持机型包括：华硕AC86U、AC68U、AX86U、AX88U、GT-AC2900等。

  2. 可以访问互联网的电脑或手机
    用于下载插件文件、访问管理后台、输入配置信息等。

  3. 一个可用的科学上网节点（代理服务器）
    可为VMess、VLess、Trojan、Shadowsocks等，需提前准备好地址、端口、UUID/密码等信息。

  4. 基础的路由器操作知识
    了解如何进入路由器管理后台、刷固件、安装插件等操作更佳。

三、安装步骤详解：从固件到插件全面部署

Step 1：刷入梅林固件

如果你的路由器尚未安装梅林固件，需要先进行刷机操作：

  1. 访问华硕梅林固件官网或Koolshare论坛；

  2. 下载与你机型对应的固件版本（注意选择稳定版）；

  3. 登录路由器后台（通常是192.168.50.1），进入**“系统管理 > 固件升级”**；

  4. 上传下载的固件文件，等待重启完成；

  5. 成功后界面左下角会显示“Merlin”字样，表明固件已正确安装。

⚠️ 刷机风险需谨慎，操作前务必备份配置，并确保电源稳定。

Step 2：安装Entware环境（插件运行基础）

  1. 登录梅林后台，进入**“软件中心”**；

  2. 安装Entware环境支持（首次使用时系统会自动提示）；

  3. 系统会挂载USB或内部存储空间作为插件依赖目录。

Step 3：安装梅林小宝插件

目前较流行的版本为Koolshare出品的科学上网插件（原梅林小宝），安装方式如下：

  1. 前往Koolshare或GitHub相关页面下载.ipk插件安装包；

  2. 登录路由器后台 → 进入“软件中心” → 选择“手动上传插件”；

  3. 选择刚下载的安装包，点击上传并安装；

  4. 安装完成后，系统将自动在“软件中心”显示科学上网插件（小飞机图标）；

  5. 点击进入插件，即可开始配置。

四、详细配置教程：让你的网络飞起来

1. 添加代理服务器节点

进入梅林小宝界面后，按照以下方式添加代理服务器：

  • 选择**“节点管理”**；

  • 点击“添加节点”；

  • 填入服务器信息（节点名、地址、端口、UUID或密码、协议类型）；

  • 可手动测试连通性并保存。

支持协议包括：

  • Shadowsocks（SS）

  • ShadowsocksR（SSR）

  • V2Ray（VMess/VLess）

  • Trojan（TLS）

2. 设置DNS加速与污染防护

科学上网中DNS设置同样关键：

  • 启用DNS防污染功能；

  • 推荐使用DoH/DoT协议的公共DNS（如Cloudflare 1.1.1.1、Google 8.8.8.8）；

  • 可开启SmartDNS插件，提升域名解析速度并实现自动分流。

3. 配置分流规则

在“访问控制”与“国外网站走代理”栏目中，你可以自由选择：

  • GFW模式：仅代理被墙网站，国内流量直连；

  • 绕过大陆IP模式：所有非中国大陆IP走代理；

  • 全局模式：所有流量均通过代理（适合测试或紧急情况下使用）；

  • 自定义ACL规则：按设备、端口、IP自定义走代理或直连。

4. 启动服务并测试连接

一切设置就绪后：

  1. 返回主界面；

  2. 点击“启动科学上网”；

  3. 页面将显示运行状态、当前流量、DNS状态等实时数据；

  4. 使用手机或电脑访问Google、YouTube等网站测试连接是否成功。

若出现连接失败，请检查网络、节点可用性、DNS设置是否冲突等问题。

五、常见问题与解决方案（FAQ）

Q1：为何配置后无法访问外网？

  • 检查代理服务器参数是否正确；

  • 是否忘记启动插件服务；

  • DNS未配置或被污染导致域名无法解析。

Q2：如何实现按设备分流？

在“访问控制”中添加MAC地址或IP地址，实现精细化管理。

例如：电视全局代理，电脑走GFW模式，打印机则完全直连。

Q3：如何自动更新规则？

插件支持定时自动更新GFW列表与路由表，建议每天更新一次，保障流畅性与有效性。

Q4：梅林小宝插件是否收费？

插件本身为开源免费项目，使用过程中无需额外费用。但代理节点（机场）多数为付费服务。

FAQ

Trojan-go 如何启用 WebSocket 支持？
在配置文件中将 transport 参数设置为 websocket，并指定 path 路径。客户端需同步使用相同路径及主机名。这样可将 Trojan 流量伪装为普通 WebSocket 通信以提升隐蔽性。
SSR 协议如何提高节点隐蔽性？
SSR 可通过选择协议和混淆方式伪装流量，降低被 DPI 检测和封锁风险。合理配置可增强节点隐蔽性和长期稳定性，保证科学上网体验顺畅可靠。
Matsuri 是否支持自动测速节点？
支持。Matsuri 可以自动对订阅中的节点进行延迟和速度测试，并按测试结果排序，用户可快速选择最佳节点，提升整体网络使用体验。
如何将 Clash 节点导入到 Sing-box？
Sing-box 支持从 Clash 配置文件中导入节点。用户只需将 YAML 文件放入配置目录或通过命令导入即可。系统会自动解析规则组和节点信息，简化迁移过程。
Nekoray 支持节点批量更新吗？
支持。Nekoray 可通过订阅 URL 批量下载最新节点并替换旧节点。结合定时更新功能，可保持节点列表最新，减少手动维护和配置工作。
Sing-box 的路由规则能否实现分国家代理？
Sing-box 支持 GeoIP 路由规则，通过引入国家数据库可根据 IP 自动判断地理位置。例如可设置 CN 走直连，US/JP 走代理，以此实现分国家策略路由，适合需要访问不同地区内容的用户。
Trojan 的伪装域名如何选择更安全？
伪装域名应选择真实可访问的网站，且证书有效。这样 Trojan 流量能伪装成正常 HTTPS 请求，减少被封锁或检测的风险，提高节点安全性和稳定性。
V2Ray 的 mKCP 参数如何调优以适应不稳定链路？
mKCP 有多个参数如 mtu、tinterval、uplink/downlink 等，可通过降低 mtu 与增大 mtu 重试间隔来适应高丢包场景；启用 FEC 与调整窗口大小能提高容错性，最终需在实际网络环境中逐步调试获得最佳值。
Sing-box 精确端口分流如何提升游戏连接？
Sing-box 可按端口设置流量分流规则，将游戏流量分配到低延迟节点。结合策略组使用，可保证游戏连接稳定，减少延迟和掉线，提高流畅度和网络体验。
Netch 全局与规则代理如何选择？
全局代理将所有流量通过节点转发，适合游戏或全局科学上网；规则代理只针对指定应用、IP 或端口走节点，可优化带宽使用和访问效率，用户可根据需求灵活选择模式。
